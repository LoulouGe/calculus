# Calcul Mental

## Projet

Site web statique pour s'entraîner au calcul mental (addition, soustraction, multiplication, division).
Créé par Louise pour son petit cousin.

## Stack

- HTML, CSS, JavaScript — pas de framework, pas de bundler
- Hébergé sur GitHub Pages

## Langues

- L'interface utilisateur est en **français**
- Le code (noms de variables, commentaires techniques) est en **anglais**
- Les messages de commit sont en **anglais**

## Conventions

- Un seul fichier HTML (`index.html`) avec CSS et JS séparés
- CSS dans `style.css`, JS dans `script.js`
- Pas de dépendances externes (pas de npm, pas de CDN)
- Code simple et lisible — la développeuse a 12 ans
